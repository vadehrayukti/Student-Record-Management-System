# Student-Record-Management-System
A simple Java-based console application demonstrating Object-Oriented Programming (OOP) concepts such as inheritance, constructors, method overriding, encapsulation, along with user input handling and dynamic data storage using ArrayList.

A simple and object-oriented Student Management System built in Java.

Features

✔ Add new students
✔ Delete students by roll number
✔ Update student details
✔ View all student records
✔ Duplicate Roll Number Prevention
✔ Demonstrates Polymorphism
